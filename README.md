# Dragon Build System

Simple build system for C and C++.

## Build
### macOS and Linux

```bash
$ sh build.sh
```

### Windows

```cmd
> build.bat
```

## Language Support

These Languages are known to work:

- C
- C++
- [Scale](https://www.github.com/StonkDragon/Scale)
- Objective-C
