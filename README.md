# Dragon Build System

Simple build system for C and C++.

## Build

```bash
$ make compile
```

After that, you should be able to do the following to rebuild the project:

```bash
$ ./build/dragon build
```

## Language Support

These Languages are known to work:

- C
- C++
- [Scale](https://www.github.com/StonkDragon/Scale)
- Kotlin (experimental)
